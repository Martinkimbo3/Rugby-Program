6-Week Pre-Season Rugby Strength Program

Progressive Overload Training System

Program Overview

Goal: Build functional strength, power, and resilience for rugby

Structure: 3 training days per week (Monday/Wednesday/Friday)

Progressive Overload: Automatic weight increases each week

Training Phases:

  • Weeks 1-2: Foundation Phase (8-12 reps) - +2.5% weekly
  • Weeks 3-4: Strength Phase (4-6 reps) - +5% jump, then +2.5%
  • Weeks 5-6: Power Phase (3-5 reps) - Maintain/slight reduction

Instructions:

  1. Enter your starting weights (in kg) for Week 1
  2. The program automatically calculates target weights for all 6 weeks
  3. Save your progress using the Save button
  4. Track actual weights lifted in the "Actual" column
